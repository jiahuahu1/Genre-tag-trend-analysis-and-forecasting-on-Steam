# Objectives
Built a quarterly analytics pipeline on Steam game data to measure tag-level supply, player attention, and review quality. <br>
Designed a trend classification framework to label game tags as Rising, Saturated, Underserved, Falling, or Stable using momentum-based metrics. <br>
Engineered forecasting models to project future tag-level release and attention trends and translated findings into market-oriented insights. <br>
